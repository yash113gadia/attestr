01 — Problem Statement

Trust in Digital Media is Broken

In an era of AI-generated deepfakes and sophisticated media manipulation, there is no reliable, accessible way to verify whether a photo or video is authentic. Journalists, legal professionals, and investigators need proof — not guesswork.

96%

Deepfakes are AI-generated

The vast majority of deepfake content is created using readily available AI tools, making manipulation trivially easy.

10x

YoY growth in deepfakes

Deepfake content is increasing exponentially, outpacing detection capabilities of traditional forensic methods.

$0

Cost to create a deepfake

Free, open-source tools make it possible for anyone to create convincing manipulated media with zero investment.

02 — Market Landscape

Existing Solutions & Their Limitations

Several players have attempted to solve media authentication, but every existing solution has critical gaps that leave users vulnerable.

Solution How It Works Blockchain? Key Limitation
C2PA / CAI Adobe, Microsoft, Google Embeds signed metadata into media files via PKI certificates No — PKI Metadata stripped by screenshots & social media uploads
Truepic $26M funded Hardware-level signing at capture (Qualcomm chips, Leica cameras) Abandoned Requires hardware integration, not retroactive
Numbers Protocol Mobile Capture App + EVM blockchain + IPFS storage Yes — EVM + NUM Requires their app; token complexity
OpenOrigins $4.5M funded Hash media → store on Hyperledger for newsrooms Permissioned Not truly decentralized
SWEAR “Digital DNA” fingerprinting per frame, stored on blockchain Hyperledger Permissioned chain, capture-forward only
Reality Defender YC-backed AI deepfake detection only No Detection only, no provenance proof
Starling Lab Stanford + USC IPFS + Filecoin storage, Hedera timestamping Yes Research framework, not a product
ProofMode Guardian Project Open-source mobile app, SHA-256 + PGP signing Optional Android only, basic UX
03 — Our Edge

What Makes MediaGuard Different

MediaGuard is the first open, browser-based media authenticator that combines blockchain provenance with forensic analysis — no hardware, no app downloads, no compromises.

🔍

Detection + Provenance

Existing tools do either AI detection or blockchain provenance. MediaGuard does both in one unified platform.

📸

Survives Screenshots

Perceptual hashing (dHash) stays intact through compression, resizing, and social media re-uploads — unlike C2PA metadata.

⛓

Truly Decentralized

Public blockchain with proof-of-work — not a permissioned Hyperledger chain controlled by a single entity.

🌐

Zero Hardware Required

Works in any modern browser. No Qualcomm chips, no special cameras, no app downloads. Just open and verify.

⏪

Retroactive Verification

Analyze existing media with forensic tools (ELA, EXIF) even if it was never registered on the blockchain.

🛡

Privacy-Preserving

Files never leave the browser. Only cryptographic hashes are stored on-chain under anonymous addresses.

08 — Roadmap

Implementation Progress

Core Blockchain Engine Complete

Block class, proof-of-work mining, chain validation, SHA-256 linking, genesis block, search by hash.

Client-Side Hashing Complete

Web Worker with SubtleCrypto SHA-256 + OffscreenCanvas dHash. Files processed entirely in-browser.

Register & Verify Flows Complete

Full upload → hash → register pipeline. Verify with exact SHA-256 match + fuzzy perceptual matching via Hamming distance.

Forensic Analysis Complete

Error Level Analysis with adjustable amplification. EXIF metadata extraction with suspicious field detection.

Camera Capture & Explorer Complete

In-browser camera with front/back toggle. Visual blockchain explorer with block inspection.

AI Deepfake Detection Next

Integrate ML-based deepfake confidence scoring using frequency-domain analysis and GAN fingerprint detection.

Public Testnet Deployment Planned

Deploy to Polygon/Base L2 for real decentralized storage. IPFS integration for optional media backup.

API & SDK Planned

Public verification API and JavaScript SDK for third-party integration into newsrooms and legal platforms.
